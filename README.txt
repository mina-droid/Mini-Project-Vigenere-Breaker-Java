PROJECT TITLE: Vigenere Breaker
PURPOSE OF PROJECT:
. Decrypt a text encrypted using Vigenere Cipher in 8 languages !    
 
VERSION or DATE: Version: 1 Date : Nov 16/2020
AUTHORS: Duke University
USER INSTRUCTIONS:

.Run breakVigenere function
.Choose the encrypted text
.Choose the languages dictionaries that you have

!!Notice: if you dont have the dictionary of the language that the text is encrypted in then you will get wrong answers
